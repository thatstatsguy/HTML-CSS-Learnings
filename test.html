Selects a subset of values falling within a specified value range.

MNSVR

Usage Example: 

Input |> SELECTVALUERANGE(Lower = 0, Upper = 10, Inverse = "False")

Arguments: 

Lower - Specifies the value range lower bound.

Upper - Specifies the value range upper bound.

Inverse - A Boolean flag that inverts the selected values. Please see the examples for more information.

Input 

1.9

 The formulae that will yield the desired result is:   

Input |> CEIL() 

The result of the function will be as follows. 

Input 

2.0
